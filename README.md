# FF_Map_Thing

## About

This project currently is in progress. Purpose is to learn pan and zoom of an image paired with an image map to have certain areas be able to be marked to keep track of information needed. It is a front-end only project.

Video Demo: https://youtu.be/RCqBqWVKNvc

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

This portion uses:<br>
javascript<br>
html<br>
css<br>
react<br>
react-scripts<br>
redux<br>
redux-thunk<br>

## How To Start

In the project directory, you can run:

### `npm install`

To ensure all the required packages are installed for this project.

### `npm start`

Runs the app in the development mode.<br>

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
